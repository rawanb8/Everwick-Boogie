# Everwick Boogee - Luxury Candles & Customization Studio

**Everwick Boogee** is a front-end project: a luxury candle shop and interactive customization studio.
This repository contains the source code for a 6-page responsive website including a home page, showcasing products, a live customization lab, a user's guide to customization, shopping/cart flow, and contact/about us and brand storytelling page

## Pages
The site contains six pages:
1. **Home** Featured collections, bestsellers, and CTAs (Call to Action)
2. **Shop** Collection browsing, product filters, and product detail pages.
3. **Candle Customization Lab** Interactive candle builder with real-time preview.
4. **User's Guide to Customization** Assist the customer in the customization process by including an ingredient description library, a personality quiz, and a random ingredient generator.
5. **Cart/Checkout** Cart mangement and checkout front-end simulation.
6. **About us/Contact us** Short storytelling of the brand plus a contact us form for questions and inquiries.

## Features
- Responsive layout
- Interactive Candle Cusotomization: choose jar, scent, label text, gift packaging and see a live preview
- Cart with persistent data using `localStorage` (for demonstration purposes only)
- Personality quiz to help guide the customer to customizing their own candle
- Smooth animations
